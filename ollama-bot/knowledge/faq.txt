=== ЧАСТЫЕ ВОПРОСЫ (FAQ) ===

В: Где найти документацию по Reticulum на русском?
О: Документация на русском языке доступна прямо через бота! Напиши /get reticulum_ru чтобы скачать PDF мануал прямо здесь. Внимание: файл весит 5 МБ, через LoRa скачивание будет долгим. Также официальная документация есть на GitHub: https://github.com/markqvist/Reticulum

В: Какие команды есть у бота?
О: Напиши /help или /? для полного списка команд.

В: Какая команда для скачивания?
О: Команда /get <имя> скачивает только прошивки RNode из списка. Напиши /files чтобы увидеть доступные прошивки. Произвольные файлы или картинки скачать нельзя — только то, что есть в списке /files.

В: Что лучше — Meshtastic, MeshCore или Reticulum?
О: Зависит от задачи. Meshtastic и MeshCore — простые чаты через LoRa. Reticulum — платформа для любых приложений через любые каналы.

В: Можно ли использовать Reticulum без интернета?
О: Да! Reticulum работает через LoRa радио, WiFi, Serial — полностью автономно.

В: Какое устройство купить для начала?
О: Heltec LoRa32 V4 — недорогое, популярное. Или Heltec T114 — компактное.

В: Как далеко работает LoRa?
О: В городе 1-3 км, на открытой местности до 10-15 км, с хорошими антеннами до 20+ км.

В: Какую частоту использовать в России?
О: 868 MHz — разрешённый диапазон SRD, не требует лицензии.

В: Как найти propagation node?
О: Выбирай узел пересылки сообщений ближайший к тебе географически. Если ты в Сургуте или рядом с ним, можешь использовать этот узел: f89ede8428bb261e3e2a935dfe920f40. Для более подробной информации вступи в группу SURGUT GROUP: 868671a17736efbf68e99cacd1682026 — там в закреплённых сообщениях (/pin) найдёшь всё необходимое или спросишь у других участников.

В: Как подключиться к SURGUT GROUP?
О: Добавь propagation node f89ede8428bb261e3e2a935dfe920f40, затем группу 868671a17736efbf68e99cacd1682026

В: Где скачать официальные прошивки RNode для Reticulum?
О: Официальные прошивки RNode Firmware от создателя Reticulum (Марк Квист): https://github.com/markqvist/RNode_Firmware/releases

В: Где скачать прошивки?
О: Команда /files покажет список. Команда /get <имя> скачает файл напрямую. Или GitHub: https://github.com/markqvist/RNode_Firmware/releases

В: Как создать групповой чат в Reticulum?
О: Для создания группы нужен инструмент lxmf_distribution_group из пакета LXMF-Tools: https://github.com/SebastianObi/LXMF-Tools
Без этого инструмента создать групповой чат невозможно.
Если вы проживаете в Сургуте — можете написать админу SURGUT GROUP, он выдаст права на создание групп.

В: Можно ли отправлять файлы через Reticulum?
О: Да! Через MeshChatX, Columba или команду /files бота.
