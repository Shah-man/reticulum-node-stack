=== СРАВНЕНИЕ MESH-СЕТЕЙ ===

== MESHTASTIC ==
- Простой mesh-чат для туристов, выживальщиков, outdoor
- Только LoRa радио
- Шифрование AES256 с общим ключом на всю сеть
- Простая маршрутизация (flooding — сообщение идёт всем)
- Приложения: чат + GPS трекинг
- Легко настроить — прошил устройство и работает
- Хорош для: походы, экспедиции, простая связь без интернета

== RETICULUM ==
- Полноценный криптографический сетевой стек
- Любые каналы: LoRa, WiFi, Ethernet, TCP, Serial
- Шифрование Curve25519 — end-to-end для каждого пользователя
- Умная маршрутизация по хешам адресов
- Много приложений: чат (Sideband, MeshChat), страницы (NomadNet), боты, файлы
- Сложнее в настройке, но гибче
- Хорош для: приватная связь, децентрализованные сервисы, mesh-инфраструктура

== MESHCORE ==
MeshCore — прошивка для LoRa устройств, альтернатива Meshtastic.
- Открытый исходный код
- Поддержка Heltec, LilyGO, RAK
- Проще и легче чем Meshtastic
- Только чат через LoRa

== ГЛАВНЫЕ ОТЛИЧИЯ ==
- Meshtastic = простой чат, Reticulum = платформа для приложений
- Meshtastic = общий ключ сети, Reticulum = персональные ключи
- Meshtastic = только LoRa, Reticulum = любые каналы
- Meshtastic проще начать, Reticulum мощнее в возможностях

== ЧТО ВЫБРАТЬ ==
- Нужен простой чат в походе? → Meshtastic
- Нужна приватность и свои сервисы? → Reticulum
- Можно использовать оба! Разные задачи — разные инструменты

== СОВМЕСТИМОСТЬ МОДУЛЕЙ ==

Meshtastic и MeshCore работают ТОЛЬКО с чипами:
- SX1262, SX1276, SX1278, SX1280

НЕ совместимы:
- E220-900M30S (чип LLCC68) — НЕ поддерживается!
- E32 модули (чип SX1278) — некоторые работают, но не все
- Любые модули на LLCC68 — не совместимы с Meshtastic/MeshCore

Для Meshtastic/MeshCore используй готовые устройства:
- Heltec LoRa32, Heltec T114
- LilyGO T-Beam, T-Deck, T-Echo
- RAK WisBlock (RAK4631)

ВАЖНО: RNode прошивка — это для Reticulum, НЕ для Meshtastic!
Не путай: Meshtastic и Reticulum — разные системы с разными прошивками.
