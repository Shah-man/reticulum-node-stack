=== КРИПТОВАЛЮТЫ И БЛОКЧЕЙН ===

== ОСНОВНЫЕ ПОНЯТИЯ ==

Блокчейн — распределённая база данных, где информация хранится в цепочке блоков.

Криптовалюта — цифровая валюта, защищённая криптографией. Работает без центрального банка.

Биткоин (BTC) — первая криптовалюта, создана в 2009. Максимум 21 миллион монет.

Эфириум (ETH) — платформа для смарт-контрактов. Вторая по капитализации.

Майнинг — процесс проверки транзакций. Майнеры получают вознаграждение за работу.

Кошелёк (Wallet) — программа для хранения криптовалюты и приватных ключей.

Приватный ключ — секретный код для доступа к криптовалюте. Никогда не делись им!

Хешрейт — вычислительная мощность для майнинга. Измеряется в H/s.

== АЛГОРИТМЫ МАЙНИНГА ==
- SHA-256 — Bitcoin
- Ethash — Ethereum Classic
- KawPow — Ravencoin
- Autolykos — Ergo
- RandomX — Monero (CPU майнинг)

== МЕХАНИЗМЫ КОНСЕНСУСА ==
Proof of Work (PoW) — майнеры решают сложные задачи. Bitcoin, Ethereum Classic.
Proof of Stake (PoS) — валидаторы блокируют монеты как залог. Ethereum 2.0, Cardano.

== ПРИВАТНЫЕ КРИПТОВАЛЮТЫ ==
Monero (XMR) — транзакции полностью скрыты, нельзя отследить отправителя и сумму.

== СВЯЗЬ С MESH-СЕТЯМИ ==
Mesh-сети и блокчейн имеют общие принципы — децентрализация, криптографическая защита.
Blockstream Satellite — транслирует Bitcoin блокчейн через спутник.
