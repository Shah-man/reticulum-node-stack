=== ЧТО ТАКОЕ RETICULUM ===

Reticulum — это криптографический сетевой стек для построения децентрализованных сетей.
Работает поверх любых каналов: LoRa, WiFi, Ethernet, интернет, serial.
Не требует центральных серверов, полностью приватный.

Установка:
pip install rns
Официальный GitHub: https://github.com/markqvist/Reticulum
Релизы: https://github.com/markqvist/Reticulum/releases
Видео от создателя (что такое Reticulum): https://youtu.be/pnyuSWy66cI

После установки доступны команды:
- rnsd — демон Reticulum
- rnstatus — статус сети и соседей
- rnpath — проверка путей до узлов
- rnprobe — пинг узлов
- rnodeconf — прошивка и настройка RNode

Частоты LoRa:
- 868 MHz — Европа, Россия (рекомендуется)
- 915 MHz — США, Австралия
- 433 MHz — любительский диапазон

Шифрование:
Всё шифруется end-to-end с помощью Curve25519.
Каждый пользователь имеет свой уникальный ключ.
Даже propagation node не может читать сообщения.

Propagation node — это узел, который хранит сообщения для офлайн пользователей.
Когда получатель появится в сети — он получит сообщения.
Как почтовый сервер, но децентрализованный.
